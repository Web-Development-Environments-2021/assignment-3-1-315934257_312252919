Coach

Up 
get /coach/{coach_name}
Returns a coach with given name (coachCoachNameGet)
Checks whether a coach exists with the given name and returns the coach object if successful. else, returns an error message.

Path parameters

coach_name (required)
Path Parameter — The name that needs to be fetched.

Return type

Coach

Example data

Content-Type: application/json
{
  "name" : "name",
  "picture" : "picture"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

the operation successful - relevant coach JSON object is fetched Coach

400

Invalid coach supplied

404

Coach not found
